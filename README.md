# Ancient Beginnings

Ancient Beginnings is a Cobblemon addon for Minecraft Forge 1.20.1 that replaces the standard starter selection with fossil Pokémon.

## Features

- Replaces the normal Cobblemon starters with fossil starters by region
- Keeps Cobblemon’s existing starter screen and starter flow
- Gives the selected starter improved starting IVs by default
- Grants a few starting items after starter selection:
    - 1 Brush
    - 10 Ancient Poké Balls
    - 3 Tumblestones
- Includes config options for:
    - minimum and maximum IVs
    - starter level
    - shiny chance
    - starter lists by region
    - starter reward items

## Included Starter Regions

- **Kanto**: Kabuto, Omanyte, Aerodactyl
- **Hoenn**: Lileep, Anorith
- **Sinnoh**: Cranidos, Shieldon
- **Unova**: Tirtouga, Archen
- **Kalos**: Tyrunt, Amaura
- **Galar**: Dracozolt, Arctozolt, Dracovish, Arctovish


## Required Dependency

Ancient Beginnings requires **Cobblemon Forge** to be installed separately.

## Installation

1. Install Minecraft Forge for 1.20.1
2. Install Cobblemon Forge 1.5.2
3. Place `ancientbeginnings-<version>.jar` in your `mods` folder
4. Launch the game

## Configuration

A common config file is generated after first launch:

`config/ancientbeginnings-common.toml`

You can adjust:
- starter IV floor and cap
- starter level
- shiny rate
- starter species lists
- starting reward items
